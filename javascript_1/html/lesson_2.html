Welcome to your first Exercise!

For this exercise we will be creating a weight metric converter

We will take ina number in pounds (lbs) and convert it to kilograms (kg). Right now we only know enough to do this one way, but later on we will learn how to add logic to this sort of problem!

To complete this exercise do the following:

  1. Link this HTML file (lesson_2.html) to tour own .js file (lesson_2.js) - ok
  2. Use Javascript to accept a number input in pounds (lbs)
  3. then in your js file convert this number to kilograms (* 0.454)
  4. Afterwards report back in an alert what the weight is in kg
  5. Then log/write "Conversion Completed" to the console
